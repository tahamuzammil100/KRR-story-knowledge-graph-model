# KRR Story Knowledge Model

An OWL ontology for modelling story plots, using the *Narn i Chîn Húrin* (Tale of the Children of Húrin) as a running example.

## Modules

| File | Prefix | Description |
|------|--------|-------------|
| `familly_relation_fo.ttl` | `fo:` | Family relationships (given) |
| `Middle_earth_ontolog_meo.ttl` | `meo:` | Middle-earth races and characters (given) |
| `rcc.ttl` | `rcc:` | Region Connection Calculus (given) |
| `place.ttl` | `npo:` | Places and spatial relations |
| `social.ttl` | `loc:` | Social relations |
| `time.ttl` | `nti:` | Time intervals and Allen's algebra |
| `event.ttl` | `nev:` | Events and their relations |
| `action.ttl` | `nac:` | Actions (intentional/unintentional) |
| `plot.ttl` | `npl:` | Plot/narrative structure |
| `narn_instances.ttl` | `narn:` | Concrete instances from the Narn |
